1. 색에 관해서

색을 나타내는 방법


색은 빨강과 초록과 파랑의 조합으로 이루어진다.




빨강과 초록과 파랑은 각각 0 ~ 255 의 값의 조합으로 만들어진다.


그리고 그 값은 16진법으로 표현된다. (0 ~ 9 a = 10, b = 11, c = 12, d = 13, e = 14, f = 15)


16진법의 표현방법은 다음과 같다.

16진법을 10진법으로 표현하는 방법은 다음과 같다.


예)7a = 7 × 16 + 10 (a = 10이기 때문)


이렇듯 16진법의 경우 두번째 자리의 숫자에 16을 곱해서 첫번째 자리 숫자에 더해주면 된다.


반대로 10진법을 16진법으로 표현하는 방법은 다음과 같다.


예) 245 → 245 ÷ 16 = 15 ... 5 → f5 (f = 15이기 때문)


이렇듯 10진법의 수를 16으로 나눈 뒤 몫을 두번째 자리의 숫자에 나머지를 첫번째 자리 숫자에 위치해두면 된다.


xxxxxxxxxx
color: red  /*색이름으로 나타내는 방법*/
color: #ffde0a /*색 빨강 = ff(255), 초록 = de(13 × 16 + 14 = 222), 파랑 = 0a(0 × 16 + 10 = 10)*/
color: rgb(255, 222, 10) /*위와 같은 색이 표시됨.*/
color: rgba(255, 222, 10, 0.2) /*위와 같은 색이지만 투명도가 20%인 경우. 원래는 투명도 100%이다.*/
/*색은 항상 0 ~ 255 사이의 수를 써야하며, rgba같은 마지막 투명도의 경우 0 ~ 1의 값을 써주면 된다.*/

2. 나머지 css 정리

나머지 우리가 배운 css 속성들을 정리해보자.


xxxxxxxxxx
color: magenta; /*글자색을 magenta로 바꾼다.*/
font-size: 10px; /*글자 크기를 10px로 바꾼다. 이외에도 pt, em 등의 단위가 있다.*/
line-height: 0; /*글자 줄 간격을 0으로 바꾼다. 이경우 글자가 완전히 겹쳐보인다.*/
font-weight: 900; /*글자 굵기를 최대 굵기로 한다. 0 ~ 900사이에 100단위 숫자를 쓴다.*/
font-style: italic; /*글자를 기울인다. 기울이지 않으려면 normal 값을 쓰면 된다.*/
text-decoration: underline; /*밑줄을 긋는다. 중첩적용할 경우 띄어쓰기를 쓰면 된다.*/
/*모든 text-decoration 속성을 없애려면 none 값을 쓰면 된다.*/
text-decoration: overline line-through underline;/*모든 선을 적용한 경우*/
background-color: rgba(0, 255, 0, 0.4) /*초록색에 투명도 40%의 배경색을 지정한 경우*/
background-image: url("google.png") /*google.png의 그림을 배경이미지로 넣는다.*/
